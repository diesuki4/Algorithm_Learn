[문제]
알파벳 소문자로 이루어진 단어가 주어진다. 
앞뒤가 똑같은 단어로 만들기 위해 문자를 바꾸려고 할 때 최소 몇 개의 문자를 바꿔야 앞뒤가 똑같은 단어가 되는지 구하라.

[입력]
첫째 줄에 단어가 주어진다. 

[출력]
첫째 줄에 바꿔야 하는 문자의 최소 개수를 출력한다.

[예제 입력 1]
racer
[예제 출력 1]
1

[예제 입력 2]
racecar
[예제 출력 2]
0

[예제 입력 3]
clear
[예제 출력 3]
2
